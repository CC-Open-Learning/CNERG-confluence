CNERG F24 : Sensors Catalog

Created by Justin Dookhran, last modified on Nov 04, 2024

Non-camera/video sensors

  • https://www.amazon.com/dp/B0CKN441VH?ref=emc_p_m_6_i

    • $160.00 + $31.31 Shipping & Import + Tax

    • Download data to phone

    • Needs to have access to wifi network with device to download data

    • 1-1.4m height placement suggested

    • Li-ion battery or USB C powered, with USB C used to recharge batteries

    • Line of sight: 1m-20m indoors, 1m-17m outdoors

    • Unable to recognize side-by-side flow

    • Exports to excel files

    • https://www.youtube.com/watch?v=1XujEOeFJSg

  • https://www.amazon.ca/Bi-Directional-Completely-Wireless-Operated-Applications/dp/B07GSG3DNN

    • $259.00 + Tax

    • 8m distance between transmitter and receiver

    • Battery operated (AA batteries)

    • Infrared line use to count people when line broken

    • Marketed as bi-directional

    • Sticker used to stick on wall/surface

    • Displays count on display on device

      • Would have to manually record in/out traffic periodically

  • https://bullseyeprotection.com/products/people-counter-bi-directional

    • Looks similar to one above (https://www.amazon.ca/Bi-Directional-Completely-Wireless-Operated-Applications/dp/B07GSG3DNN )

  • https://www.amazon.ca/dp/B07QSFBJ1J/ref=sspa_dk_detail_0?s=industrial&sp_csd=d2lkZ2V0TmFtZT1zcF9kZXRhaWw

    • $659.95 + Tax

    • Free app available on android/iPhone

      • Requires account to be created

      • App collects personal data (name, email, address, phone number) on android (SMS Storetraffic))

      • Multiple sensors per location supported

      • Gives hourly breakdown of people counting

      • TMAS website can be linked to app

        • https://www.smssoftware.net/tmas/index.jsp

    • Data updates every hour

    • Stick and peel to mount on wall/surface

    • Connects using wifi

    • Configured using mobile app

    • Uses AA batteries

  • https://www.amazon.ca/People-Counter-New-Data-Statistics/dp/B0C9V588ZQ?th=1

    • $329.00 + Tax

    • Data recorded in CSV format

    • Uses micro SD cards

    • Records beam interruptions to count people

      • Stores beam interruption records over past 180 days

        • Data is stores in memory in device

    • Uses AA batteries that can last up to 12 months

  • https://www.amazon.ca/dp/B07CJJ49F9

    • $395.00 + Tax

    • Mounted using self-tapping fasteners or adhesive tape

    • Uses AA batteries

    • Uses micro SD card

    • Internal data storage in CSV format

    • Display on device can be turned off to hide count numbers

    • Can store data on device for last 180 days

  • https://www.footfallcam.com/en/Product/FootfallCam-Space-Sense

    • Passive infrared sensors used to detect occupancy of a facility

    • Uses CR2450 batteries

    • Works in inconsistent lighting situations

  • https://iee-sensing.com/smart-buildings/people-counting/people-counter/

    • Bi-directional counting

    • Application that can keep track of time people spend in spaces

  • https://www.eco-counter.com/produits/pyro-evo-range-en/pyro-boxevo/

    • Offices in Montreal, Quebec

    • Data is collected manually using Eco-Link software and magnetic key

    • Bi-directional counting

  • https://www.trafx.net/products#vehicle-counter

    • Canadian company

    • Not bi-directional counting

    • Good for outdoors

    • Detection range 6m

    • -40c to 55c

    • 3x AA batteries with 9-10 year life

    • 1 year limited warranty covering manufacturing defects

    • Exports data to .txt file

  • https://dingtek.com/dc520-bi-directional-people-counter

    • $102 USD for multi-tech version, $85 USD for LoRaWAN version

    • Wave radar sensing

    • Bi-directional

    • Multiple transmission options

    • Should be able to connect data to other platforms

      • $10/year fee to use their application

    • Export data to CSV or excel format

    • Micro USB powered

    • Multiple wireless bands

    • Detection Range <= 3m

    • Temperature Range -20c - 75c

  • https://sensmax.eu/devices/wireless-bidirectional-people-counting-sensor-sensmax-pro-d3-superlongrange/

    • Infrared ray technology

    • Creates invisible line that counts when line is broken

    • Side by side traffic (people passing beside each other) may be counted as one person

    • Sensors that can be installed at 2.5m-3m height to count people walking by
