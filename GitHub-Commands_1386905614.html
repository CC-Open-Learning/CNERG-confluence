CNERG F24 : GitHub Commands

Created by ASHISH GYAWALI, last modified on Nov 06, 2024

// For new projects 

 

# 1. Initialize a Git repository

git init

 

# 2. Add the Bitbucket repository as a remote origin

git remote add origin{your repo link}

 

# 3. Create the {branch-name} branch locally

git checkout -b {branch-name}

 

# 4. Stage all your files

git add .

 

# 5. Commit the staged changes

git commit -m "Initial commit"

 

# 6. Push your changes to the {branch-name} branch of the Bitbucket repository 

git push -u origin {branch-name}

 

// For existing projects

 

# 1. Add changes to the staging area
git add .

 

# 2. Commit the changes with a message
git commit -m "Your commit message"

 

# 3. Fetch the latest changes from the remote repository
git fetch

 

# 4. Merge the fetched changes into your current branch (from origin/{branch-name} or the appropriate branch)
git merge origin/branch-name

 

# 5. Pull any new changes from the remote and update your local branch
git pull origin branch-name

 

# 6. Push your local changes to the remote repository
git push origin branch-name

If any conflicts on local and git:

git fetch origin

git rebase origin/{branch-name}

git push origin {branch-name}
