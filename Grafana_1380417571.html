CNERG F24 : Grafana

Created by Eunie on Oct 04, 2024

Grafana is an open-source platform for visualizing and monitoring data. It integrates with various databases, allowing users to create dashboards and analyze data in real time. Grafana is widely used for tracking system performance, visualizing time-series data, and setting up alerts when specific conditions are met.

Key Features of Grafana:

  1. Data Visualization: Grafana allows you to create dashboards with different types of charts, graphs, and tables to visually represent your data.

  2. Alerting: You can set up alerts based on predefined conditions, which notify you through email, Slack, or other channels when triggered.

  3. Plugins: It supports many plugins for integrating with popular data sources such as InfluxDB, MySQL, PostgreSQL, Prometheus, and more.

  4. Dashboard Sharing: Grafana allows users to easily share dashboards with others or embed them into websites.

Install Grafana (on Windows):

  1. Download Grafana:

    • Go to the Grafana Download page and download the installation package for your operating system (for Windows, download the MSI installer).

    • https://grafana.com/grafana/download

  2. Install Grafana:

    • Run the downloaded installer and follow the on-screen instructions to install Grafana.

    • By default, Grafana will be installed in the C:\Program Files\GrafanaLabs\grafana directory.

  3. Run Grafana:

    • Once installed, Grafana will automatically start as a service.

    • Open a web browser and go to http://localhost:3000 to access the Grafana web interface.

    • The default login credentials are:

      • Username: admin

      • Password: admin (You will be prompted to change the password upon first login).
