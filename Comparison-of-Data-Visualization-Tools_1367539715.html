1

Power BI

Method 1) Power BI REST API

  • Free for personal use

  • Requires Pro license for sharing or team collaboration

  • REST API available for data fetching

  • Personal dashboards can be embedded via WebView

Best for personal real-time data visualization, but requires Pro for collaboration

Free for personal use, Power BI Pro required for team sharing (Paid subscription)

Method 2) Power BI Embedded

  • Designed for embedding dashboards in third-party apps

  • REST API & iframe embedding

  • Full embedding and customization capabilities for enterprise use

Best for secure, fully embedded enterprise-level dashboards

Requires Power BI Embedded plan, costs based on usage (Paid)

2

Grafana

  • Optimized for real-time monitoring

  • Supports various data sources

  • Can be embedded via iframe using WebView

  • Provides real-time sensor data visualization

Best for real-time data visualization and monitoring in Unity

Free (Open Source), Paid options available for advanced features

3

Google Data Studio

  • Easy to use

  • Integrates well with Google products

  • Free to use

  • Can be embedded via iframe using WebView

  • Slight delay in real-time data updates

Best for non-real-time data visualization or simple dashboards

Free

4

Metabase

  • Open-source

  • Integrates with various data sources

  • Self-hosted or cloudMetabase

  • Can be embedded via iframe using WebView

  • REST API available for data fetching and custom visualization

Best for periodic data visualization, not real-time

Free (Open Source), Paid cloud-hosted options available

5

Apache Superset

  • Optimized for advanced data visualization

  • Supports complex dashboards

  • Can be embedded via iframe using WebView

  • REST API available for data fetching

Best for advanced data visualization or complex dashboards

Free (Open Source)

6

Tableau Public

  • Strong visualization capabilities

  • Free version available

  • Can be embedded via WebView, but data must be public

Best when data can be publicly displayed and strong visualization is required

Free (Public version), Tableau Desktop (Paid)
